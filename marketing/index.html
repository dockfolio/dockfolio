AI-Native

Built for developers who use AI

Dockfolio is designed to work seamlessly with AI coding tools like Claude Code. Every feature is API-accessible, scriptable, and automatable.

144 REST Endpoints Every dashboard feature is an API call. Claude Code can manage your infrastructure, check revenue, restart containers, and run security audits — all through natural language.
AI Morning Briefing Automated daily intelligence report covering system health, overnight events, revenue changes, and action items. Powered by Anthropic Haiku. Cost: ~$0.001/day.
Command Palette Ctrl+K fuzzy search across apps, commands, and actions. Same keyboard-first UX you know from your editor and terminal.
Self-Hosted Data Your metrics, revenue, errors, and traffic data stay on your server. Feed it to AI tools without sharing with third-party SaaS platforms.
Built Different

The first ADHD-friendly DevOps dashboard

Built by a developer with ADHD who got tired of noisy dashboards with too many blinking lights. Press A to enter a calmer, focused experience.

Kill the Noise Dims healthy items. Stops all animations. Boosts contrast. You only see what needs attention.
Focus Mode Shift+Z strips the dashboard to a single app. Zero distractions. One thing at a time.
Streak Bar Gamified uptime tracking. Maintain your incident-free streak. Small dopamine hits for keeping things healthy.
